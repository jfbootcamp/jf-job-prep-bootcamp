1. Java 기본 타입
    1) 숫자 타입 선택 
        - 일반적인 정수 => int 
            - 대부분의 경우 int로 충분
            - 루프 카운터, 배열 인덱스, 일반 계산 
        - 큰 숫자가 필요할 때 => long 
            - 파일 크기
            - 전 세계 유저 수 
            - 금용 시스템의 금액
        - 실수가 필요할 때 => double
            - 확률, 비율, 평균
            - 좌표 (위도/경도)
            - 과학 계산 
        - 정밀한 금융 계산 => BigDecimal
            - 이자 계산
            - 환율 계산
            - 소수점 정확도가 중요한 모든 경우    

    2) 숫자 타입의 범위

            타입        크기        범위                실무 사용 예
          --------------------------------------------------------
            byte        1B       -128~127             거의 안 씀   
            short       2B       -32,768~32,767       거의 안 씀  
            int         4B        +- 21억             기본 선택 
            long        8B        +- 922경             큰 ID 

            float       4B         7자리 정밀도         거의 안씀       
            double      8B        15자리 정밀도         실수 기본 선택
