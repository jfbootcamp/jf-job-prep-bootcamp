    1) 깃에서 버전을 만드는 단계마다 파일 상태가 달라짐.

2. tracked 파일 vs untracked 파일     
    1) Changes not staged for commit
        - 변경된 파일이 아직 스테이지에 올라가지 않았다는 뜻임.
        - modified: (수정되었다는 것임)
        - 깃은 한번이라도 커밋한 파일은 계속해서 수정 사항이 있는지 추적함 
        - tracked 파일임. 
            - 깃이 추적하고 있다는 뜻임.
    2) Untracked files:
        - 한번도 커밋하지 않았으므로 수정 내역을 추적하지 않았음.
