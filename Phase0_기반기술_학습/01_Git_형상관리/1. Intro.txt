1. 깃 (Git)
    1) 2005년 리누스 토르발스가 깃을 소개 
        - 리눅스 창시자 
    2) 깃으로 만들어진 소프트웨어가 리눅스(Linux) 운영체제임
        - 깃을 통해 리눅스의 수많은 소스 코드를 효율적으로 관리

2. 깃 핵심 기능
    1) 버전 관리하기 
    2) 백업하기 
    3) 협업하기         

3. 깃 환경 설정하기
    1) 사용자 정보 입력해야 함 
        - 깃은 버전을 저장할 때마다 그 버전을 만든 사용자 정보도 함께 저장하기 때문임.
        - git config --global user.name jfbootcamp     
        - git config --global user.email jfbootcamp@gmail.com
        - git config --list

4. 리눅스 명령 연습
    1) pwd  
        - 현재 위치의 경로가 나타남 
        - print working directory       

    2) ls         
        - 현재 디렉터리에 어떤 파일이나 디렉터리가 있는지 확인할 때 쓰는 명령어 
        - 디렉터리와 파일 이름이 나타남 
        - 이름 뒤에 슬래시(/)가 붙어 있으면 디렉터리임.
        - ls -l 
            - 파일이나 디렉터리의 상세 정보까지 표시함 
          ls -a
            - 숨김 파일이나 디렉터리도 함께 표시함. 
          ls -r
            - 파일의 정렬 순서를 거꾸로 표시함.
          ls -t
            - 파일 작성 시간순(내림차순)으로 표시함.

    3) clear
        - 터미널 창 지우기 

    4) cd
        - 터미널 창에서 디렉터리 이동하기 
        - change directory
        - cd ..
            - 상위 디렉터리로 이동함
