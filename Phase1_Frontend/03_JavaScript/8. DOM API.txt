1. DOM
    1) 자바스크립트는 웹 페이지를 동적으로 만들어 주기 위해 개발된 프로그래밍 언어임
    2) 어떻게 자바스크립트가 웹 페이지를 이루는 요소들을 접근하고, 어떻게 웹 페이지를 
       동적으로 만들 수 있는지 확인할수 있는 객체임. 
    3) 자바스크립트는 DOM을 통해 웹 페이지의 요소에 접근 할수 있음 
    4) DOM 
        - Document Object Model
        - HTML로 작성된 여러 요소들에 자바스크립트가 접근하고 조작할수 있도록
          브라우저가 변환시킨 객체임.
        - HTML을 객체(Object)로 표현한 트리 구조 
        - JavaScript가 웹 페이지를 동적으로 조작하는 핵심 인터페이스임.

2. 노드(Node)
    1) 태그, 텍스트, 속성이 각각 노드가 됨 
    2) 종류 
        - 요소 노드 
        - 텍스트 노드 
        - 속성 노드 

3. 왜 DOM 트리로 변환하는가?
    1) JavaScript가 HTML을 조작할 수 있게 함
    2) 브라우저가 화면을 그리는 기준이 됨 
    3) HTML 문서 -> DOM 트리 변환은 브라우저가 자동으로 수행함
