1. 배열
    1) 객체가 한 번에 서로 연관된 여러 데이터의 값을 저장할수 있는 자료형이었다면,
       배열은 순서가 있는 요소들의 집합 
    2) 여러 개의 데이터의 모여있는 리스트라고 할수 있음 
    3) 자료구조  
    4) 순서(인덱스)로 데이터 관리 --> [값1, 값2, 값3]

2. 배열 생성

3. 배열 요소 

4. 배열 요소 접근 

5. 배열 요소 추가 (메서드)
    1) push() : 배열의 마지막 인덱스에 요소 추가 
    2) unshift() : 배열의 첫 번째 인덱스에 요소 추가 

6. ... (스프레드 연산자)
    1) 배열이나 객체 안의 요소를 하나씩 펼쳐서 복사하거나 전달할 때 사용하는 문법 
    2) 묶여 있는 것을 펼침 
    3) 원본 배열 보호 (복사)
        - 원본 배열이 아닌, 복사본을 출력하거나 사용하기 위해서임     

7. 배열 요소 수정 

8. 배열 요소 삭제
    1) pop() : 배열의 마지막 요소 삭제 
    2) shift() : 배열의 첫번째 요소 삭제 
    3) splice() : 배열의 특정 인덱스의 요소 삭제
