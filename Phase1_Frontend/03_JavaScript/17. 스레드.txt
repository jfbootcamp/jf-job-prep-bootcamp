1. 스레드 (Thread)
    1) 프로세스가 작업(Task)을 실행하는 "작업 단위"
    2) 비유
        - Single Thread 
            - 요리사 1명 ---> 한 번에 하나의 요리만 가능 
            - 작업이 순차적으로 실행됨 
            - JavaScript, Python 등 
        - Multi Thread 
            - 요리사 여러 명 --> 동시에 여러 요리 가능 
            - 작업이 병렬로 실행됨 
            - Java, C++, Go 등        

2. 왜 JavaScript는 싱글 스레드일까?
    1) 웹 브라우저 환경에서 탄생 
        - 1995년 넷스케이프에서 10일 만에 개발 
        - 단순히 웹 페이지 조작이 목적이었음
    2) DOM 조작의 안정성
        - 두 스레드가 동시에 같은 DOM을 수정하면?
        - 충돌 발생! (Race Conditon) 
        - 싱글 스레드면 이 문제가 원천 차단됨
    3) 단순함과 예측 가능성
        - 코드 실행 순서가 명확함
        - 디버깅이 상대적으로 쉬움 
    4) 하지만 비동기로 "동시에 처리하는 것처럼" 동작         

3. 싱글 스레드의 한계를 극복하는 방법
    1) 이벤트 루프를 통해 싱글 스레드의 한계를 극복 
        - 자바스크립트는 싱글 스레드 방식으로 한 번에 하나의 작업만 처리 할수 있음
        - 그러나 브라우저의 Web APIs와 이벤트 루프를 통해 마치 멀티 스레드처럼 
          비동기적으로 처리 가능함  

    2) Web APIs 
        - 브라우저가 제공하는 기능 (자바스크립트 엔진 외부에 위치함)
        - 자바스크립트가 싱글 스레드의 한계를 극복하도록 비동기 작업(타이머, 
          HTTP 요청, DOM 이벤트 등)을 대신 처리해 줌.
        - 작업 완료 후 그 결과로 실행될 콜백 함수를 태스크 큐에 추가하여 
          이벤트 루프가 처리하도록 전달함.    

    3) 태스크 큐
        - 비동기 함수의 콜백 함수 또는 이벤트 핸들러가 실행되기 전에 일시적으로 
          대기하는 공간 
        - JavaScript 엔진의 콜 스택이 비어 있을 때 태스크 큐에 있는 작업들이
          콜 스택으로 이동하여 실행됨   

    4) 이벤트 루프 
        - 콜 스택과 테스크 큐를 계속해서 모니터링하며 작업을 관리하는 루프 
        - 콜 스택이 비워 있을 때 테스크 큐에 대기중인 함수가 있으면,
          이벤트 루프는 순차적으로 태스크 큐에서 콜백 함수를 콜 스택으로 이동시켜 실행함.
        - 이를 통해 자바스크립트가 싱글 스레임에도 마치 여러 작업이 동시에 처리되는 것처럼
          보이게 함                                                                 

4. 정리
    1) 브라우저는 멀티 스레드 방식으로 동작함 
    2) JavaScript 엔진(콜 스택과 힙을 관리)은 싱글 스레드로 동작하지만,
       이는 JavaScript 엔진만의 특징임
    3) 비동기 작업(타이머, HTTP 요청, DOM 이벤트 등)은 브라우저의 Web APIs가 
       별도의 멀티 스레드 환경에서 처리함.
    4) 따라서 JavaScript 코드 자체는 싱글 스레드로 동작하지만, 
       브라우저 환경 전체는 멀티 스레드로 동작하기 때문에 비동기 처리가 가능함 

    5) JavaScript 엔진 : 싱글 스레드 
       브라우저(Web APIs, 이벤트 루프, 랜더링): 멀티 스레드 
       이러한 구조 덕분에 JavaScript가 싱글 스레드임에도 불구하고 비동기 처리가 가능함
