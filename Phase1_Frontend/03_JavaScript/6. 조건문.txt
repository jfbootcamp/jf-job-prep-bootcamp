1. 조건문 
    1) 특정 조건에 따라 코드를 실행하거나 실행하지 않을 때 사용하는 구문

2. if문 
    1) 가장 일반적으로 사용되는 조건문     

3. if-else문 
    1) 조건문을 만족할 때와 만족하지 않을 때 각각 다른 코드를 실행 시키기 위함
    2) '만약 ~하면 ~하고 그렇지 않다면 ~하라'    

4. 중첩 조건문 
    1) if 안에 if 
    2) 문법
        if(외부조건) {
            if(내부조건) {
                // 둘 다 만족                    
            } else {
                // 외부만 만족    
            }
        } else {
            // 외부조건 불만족                
        }

5. if-else if문 
    1) 여러 조건에 따라 서로 다른 코드를 실행하야 할 때 사용되는 조건문 
    2) 다중 조건문 

6. switch/case문 
    1) 주로 실행할 코드를 여러 조건으로 나눠야 할 때 사용함 
    2) 기본구조

        switch(변수) {
            case 값1:
                // 변수 === 값1 일 때 실행 
                break;      // 여기서 switch 종료!
            case 값2:
                // 변수 === 값2 일 때 실행 
                break;           

                ....
            default:  
                // 모든 case에 해당 안 될 때              

        }

        - break 없으면 다음 case도 실행함!
