1. 객체 (Object)
    1) 비 원시 타입 자료형에 해당하는 자료형 
    2) 한번에 여러 개의 데이터 값을 저장할 수 있는 자료형 
    3) 예
        - 사람(person) 
            -  나이, 이름, 전화번호, 거주지,....
    4) 키(key)와 값(value)의 쌍으로 데이터를 저장하는 자료구조 
    5) 이름(키)으로 데이터 관리 --> {키1: 값1, 키2: 값2}            

2. 객체 생성
    1) 생성자 함수를 사용하는 방법 
    2) 리터럴을 사용하는 방법 

3. 객체 프로퍼티

4. 객체 프로퍼티 추가, 수정, 삭제 

5. Object 클래스 
    1) JavaScript의 모든 객체의 최상위 부모(조상) 클래스
    2) 모든 객체는 Object를 상속받음 
    3) { } -> Object (프로토타입 체인)
    4) Object 클래스가 제공하는 것
        - 정적 메서드 : keys(), values(), entries()
            - Object.keys() : 객체의 키를 배열로 반환 ["name","age"]
            - Object.values() : 객체의 값을 배열로 반환 ["이순신", 25]
            - Object.entries() : [키,값]쌍 배열을 반환 [["name","이순신"],["age", 25]]
            - 객체는 배열이 아니라서 for문으로 직접 순회할수 없음.
                - Object.keys/values/entries로 배열로 변환 --> for문으로 순회 가능

6. 객체(Object) vs Object 클래스
    1) 객체 (object) - 소문자 
        - 데이터를 담는 그릇 (인스턴스)
        - 우리가 만드는 개별 객체 
    2) Object 클래스 - 대문자 
        - 객체를 만들고 다루는 도구 
        - JavaScript가 제공하는 내장 클래스
