1. 자료형(Type)
    1) 변수에 할당되는 데이터의 종류를 뜻함 
    2) 변수의 타입 
    3) 원시 타입 vs 비원시 타입 
    4) 원시 타입
        - 숫자, BigInt, 문자, boolean, null, undefined
    5) 비원시 타입
        - 객체, 함수, 배열이 포함된 객체 
    6) typeof 
        - 특정 변수의 자료형을 반환하는 연산자임     

2. 숫자형 
    1) 숫자를 나타내는 자료형, number 
    2) 정수, 소수 등의 모든 숫자를 나타냄 
    3) Infinity, NaN과 같은 특수한 값도 포함됨 
    4) Infinity
        - 무한대를 나타내는 값    
    5) NaN 
        - Not A Number 라는 뜻임.
        - '숫자가 아니다'라는 뜻임.   

3. BigInt
    1) 2^53-1보다 크거나, -(2^53-1)보다 작은 정수를 나타내야 할때 사용하는 자료형.
    2) BigInt형은 숫자 끝에 n을 붙임
    3) BigInt()라는 함수를 사용해 나타낼수 있음 

4. 문자형 
    1) String 타입 
    2) 변수에 문자열을 할당하기 위해서는 문자열을 따옴표로 묶어야 함 
    3) 큰따옴표(""), 작은따옴표(''), 역따옴표(``)의 3가지 종류가 있음 
    4) 역따옴표(``)
        - 문자열 안에 ${}안에 작성된 변수에 할당된 값이 문자열과 함께 출력됨
        - 템플릿 리터럴 
    5) 권장
        - 일반 문자열은 작은따옴표, 변수 삽입 시 백틱 사용 

5. boolean 형 
    1) 참이나 거짓을 표현하기 위한 자료형 
    2) 값이 true 또는 false로 이루어져 있음 
    3) 보통 참인지 거짓인지를 분별하는 값을 저장할 때 사용 됨 
    4) 조건문을 사용해 변수의 값에 따라 다른 코드를 실행해야 할 때 자주 사용되는 자료형 
    5) Falsy(6가지)
        - false, 0, '', null, undefined, NaN
    6) Truthy
        - 나머지 전부 ([], {}, 'false')   

6. null 형 
    1) null을 하나의 값으로만 활용하는게 일반적인 언어에서 활용임 
    2) 자바스크립트에서 null은 오직 null값만 포함하는 자료형으로 사용됨 
    3) null 값 
        - 존재하지 않거나 알수 없는 값을 나타날 때 사용됨 

7. undefined 형 
    1) null 형과 자주 햇갈릴 수 있는 자료형임 
    2) undefined값만을 포함하는 자료형 
    3) 변수에 값이 할당되지 않은 상태일 때 자동으로 undefined 값이 할당되는 차이가 있음.
