1. 자바스크립트 엔진 
    1) 자바스크립트는 JS 엔진이 있는 모든 곳에서 동작함.
    2) 자바스크립트 코드를 실행하는 프로그램 
    3) 우리가 일반적으로 사용하는 웹 브라우저 
    4) 가장 대표적인 엔진은 크롬 브라우저의 V8 엔진임.

2. 웹 브라우저에서 실행 
    1) 자바스크립트 엔진이 들어 있는 웹 브라우저를 통해 자바스크립트를 직접 사용함 

3. 변수
    1) 프로그램이 실행되는 도중에 변경되는 값을 저장하기 위한 '이름을 가진 저장소'임.
    2) 변수를 사용하기 위해서는, 우선 변수 이름을 지정해야 함.
    3) 변수에 이름을 지정할 때 let이라는 키워드를 사용함.
        - let 뒤에는 생성할 변수 이름을 작성함   

4. 변수 명명 규칙 
    1) 변수 선언이라 부름
    2) 변수명에는 $ 기호와 _기호를 제외한 다른 특수 기호는 사용할 수 없음
    3) 변수명의 맨 앞에는 숫자를 사용할 수 없음
    4) 변수명에는 예약어를 사용할 수 없음 

5. 긴 변수명 표기법
    1) 변수의 이름을 되도록 변수가 담고 있는 값이 무엇인지 혹은 변수 역할이 무엇인지를
       명확하게 알 수 있게 지정함.
    2) 변수명에 두가지 이상의 단어가 들어가야 해서 변수명이 길어지게 되는 경우가 있음 
    3) 카멜 표기법(Camel case)       
        - 두가지 이상의 영단어로 이루어진 변수명을 작성할 때,
          가독성을 높이고 단어를 한눈에 파악할 수 있게 도와주는 표기법 
        - 단어를 차례대로 나열하면서 첫번째 단어를 제외한 각 단어의 첫글자를 대문자로
          작성하는 방법임.     

6. 상수 
    1) 변하지 않는 값을 저장하는 자정소 
    2) 상수를 선언할 때에는 let이 아닌 const 키워드를 사용함 
    3) 상수에 한 번 저장한 값은 변경할 수 없음       
    4) 실수로 변수의 값이 변경되는 것을 방지하기 위해 사용됨 
    5) 이메일이나 홈페이지 주소와 같이 특정 값을 저장해두는 별칭으로 사용하기도 함
        - 상수를 별칭으로 사용할 때는, 일반적으로 상수의 이름을 모두 대문자로 작성함
