CSS 박스 모델 완벽 가이드

웹 페이지의 모든 요소는 박스(Box)로 이루어져 있습니다. 이 박스들이 어떻게 구성되고, 서로 어떻게 배치되는지 이해하는 것이 CSS 레이아웃의 핵심입니다.

본문 영역의 너비가 너무 넓으면 한 줄에 너무 많은 글자가 들어가서 읽기 불편합니다. 일반적으로 한 줄에 45~75자 정도가 가장 읽기 편하다고 알려져 있습니다. 그래서 max-width로 너비를 제한하고 중앙에 배치하는 것이 좋습니다.
