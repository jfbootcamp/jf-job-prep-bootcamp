CSS 박스 모델의 이해

웹 페이지를 구성하는 모든 HTML 요소는 박스(Box) 형태로 표현합니다. 이 박스는 콘텐츠(content), 패딩(padding), 테두리(border), 마진(margin)으로 구성되어 있습니다.

위 그림에서 볼 수 있듯이, 가장 안쪽에 실제 콘텐츠가 위치하고, 그 바깥으로 padding, border, margin이 차례로 감싸고 있습니다. 이 구조를 이해하면 요소의 크기와 간격을 정확하게 제어할 수 있습니다.

특히 box-sizing: border-box를 사용하면 with에 padding과 border가 포함되어 계산이 훨씬 직관적이 됩니다.
