1. position: relative와 absolute 관계
    1) 핵심개념
        - position: absolute는 가장 가까운 positioned 조상을 기준으로 위치가 정해짐 
        - 부모에 position: relative가 없으면 ==> 뷰포트(화면) 기준으로 이동
        - 부모에 position: relative가 있으면 ==> 부모 기준으로 위치 지정 

2. ::after 가상 요소 
    1) 가상 요소 (Pseudo-element)는 HTML에 직접 작성하지 않아도 
       CSS로 생성할 수 있는 '가상의 요소'임.         
       - ::before     요소의 내용 '앞'에 가상 요소 삽입   
       - ::after      요소의 내용 '뒤'에 가상 요소 삽입

    2) CSS 가상 요소와 position 정리
        - ::after 가상 요소 
            - CSS로 HTML 없이 요소의 컨텐츠 뒤에 가상 콘텐츠를 생성
            - content 속성이 필수 (없으면 랜더링 안됨)     
            - DOM에 존재하지 않음 (JavaScript로 직접 접근 불가)
            - 장점
                - HTML이 깔끔해짐
                - CSS로 일괄 관리 가능 
                - 그라데이션 + 애니메이션 효과 적용 가능 
        - ::before 가상 요소                
            - CSS로 HTML 없이 요소의 컨텐츠 앞에 가상 콘텐츠를 생성
            - ::after와 동일하게 content 속성 필수 

        - ::after vs ::before 차이 

            가상요소        위치        사용 예시 
           ------------------------------------------------
            ::before     컨텐츠 앞     리스트 아이콘, 인용부호                 
            ::after      컨텐츠 뒤     밑줄, 장식 요소  

        - position 핵심개념 
            - position: absolute는 가장 가까운 positioned 조상을 기준으로 위치가 정해짐 
            - 부모에 position: relative가 없으면 --> 뷰포트(화면) 기준으로 이동
            - 부모에 position: relative가 있으면 --> 부모 기준으로 위치 지정
