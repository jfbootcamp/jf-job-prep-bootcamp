1. grid-template-rows 
    1) 명시적 행(Track)의 크기를 정의함 
    2) fr 단위 지원
    3) repeat() 함수 
    4) 라인 이름 정의 가능함 
    5) 각 라인에는 자동으로 숫자 이름이 부여됨 
        - 1, 2, 3... 또는 -1, -2, -3 ...

2. grid-template-columns
    1) 명시적 열(Track)의 크기를 정의함 
    2) fr(fraction) 단위 지원
        - 남은 공간을 비율로 배분함 
        - 컨테이너가 가변해도 열 크기가 자동 조절됨 
    3) repeat() 함수 
    4) 라인 이름 정의 가능함    

3. Named Grid Lines (이름 있는 그리드 라인)    
    1) 숫자 대신 의미 있는 이름으로 아이템을 배치함 
        - 가독성 향상, 유지보수 용이, 실무 필수 
    2) 기본 문법
        - 라인 이름 정의 방법
            - [이름]  <---- 대괄호 안에 이름 작성 
        - .container {
            grid-template-columns: 1fr [center-start] 1fr [center-end] 1fr     
          }
