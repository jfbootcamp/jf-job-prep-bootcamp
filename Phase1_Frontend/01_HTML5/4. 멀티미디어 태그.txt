1. video 태그
    1) HTML5에서 도입된 비디오 재생 태그 
    2) 플러그인(Flash 등)없이 브라우저에서 직접 재생 
    3) 대부분의 최신 브라우저에서 지원 

2. video 태그 주요 속성 
    1) controls
        - 재생/일시정지, 불륨, 진행바, 전체화면 등 컨트롤 UI 표시 
        - 이 속성이 없으면 사용자가 비디오를 조작할 수 없음 
    2) poster
        - 비디오 재생 전에 보여줄 이미지 
        - 사용자가 재생 버튼을 누르기 전까지 표시됨
    3) preload
        - 페이지 로드 시 비디오를 얼마나 미리 로드할지 결정 
        - auto : 전체 비디오 미리 로드 (데이터 사용량 많음)
        - metadata : 메타데이터(길이, 크기 등)만 로드 (권장) 
        - none : 아무것도 로드하지 않음 (재생 시 로드)

3. source 태그 
    1) 하나의 video 태그에 여러 형식의 파일을 제공 
    2) 브라우저는 위에서부터 순서대로 지원하는 형식을 찾아 재생 
    3) type 속성으로 MIME 타입을 명시하면 브라우저가 더 빨리 판단        
    4) 비디오 형식별 특징 
        - MP4 (video/mp4)
            - 가장 널리 지원되는 형식 
            - H.264 코덱 사용 
            - 모든 주요 브라우저에서 지원 
            - 권장: 첫 번째 source로 배치 
        - WebM (video/webm)
            - Google이 개발한 오픈 형식
            - VP8/VP9 코덱 사용 
            - Chrome, Firefox, Edge에서 지원
        - Ogg (video/ogg)
            - 오픈 소스 형식
            - Theora 코덱 사용
            - Chrome, Firefox에서 지원
            - 현재는 잘 사용하지 않음                         
    5) src : 비디오 파일 경로 



4. audio 태그 
    1) HTML5에서 도입된 오디오 재생 태그 
    2) 플러그인(Flash 등)없이 브라우저에서 직접 재생   
    3) 대부분의 최신 브라우저에서 지원 

5. audio 태그 주요 속성  
    1) loop 
        - 배경음악처럼 계속 재생해야 할 때 사용
        - 오디오가 끝나면 자동으로 처음부터 다시 재생
        - 사용자가 직접 정지하기 전까지 무한 반복 
        - preload="auto" 사용         
            - 반복 재생할 오디오는 미리 전체를 로드하는 것이 좋음
            - 재생 중 끊김 방지 
    2) autoplay
        - 페이지 로드 시 자동 재생
    3) muted
        - 음소거 상태 (autoplay와 함께 필수!)    

6. YouTube 임베드 
    1) YouTube 동영상을 웹페이지에 삽입하는 방법
        - 외부 웹사이트에 YouTube 동영상을 삽입하는 것 
        - iframe 태그를 사용하여 구현 
    2) iframe 태그의 기본 사용법 습득
        - 주요 속성 
            - src : 임베드할 콘텐츠의 URL 
                - https://www.youtube.com/watch?v=b7k0a5hYnSI
                - https://youtu.be/b7k0a5hYnSI
                - https://youtu.be/b7k0a5hYnSI?si=OPPO3jsesiZTzVqc
            - title : 접근성을 위한 제목 (스크린 리더용)
            - allow : 허용할 기능들 
                - 속성 값들 
                    - accelerometer : 가속도계 센서 접근 
                    - autoplay : 자동 재생 허용          
                    - clipboard-write : 클립보드 쓰기 
                    - encrypted-media : 암호화된 미디어 재생(DRM)
                    - gyroscope : 자이로스코프 센서 접근 
                    - picture-in-picture : PIP 모드 허용
            - allowfullscreen : 전체화면 허용                           
    3) YouTube URL 파라미터 활용법 습득
        - 재생 관련
            - start=초: 해당 초부터 시작 (예: start=30 => 30초부터)
            - end=초: 해당 초에서 종료
            - autoplay=1: 자동 재생 (mute=1 필요)
            - mute=1: 음소거 상태로 시작
            - loop=1: 반복 재생
            - rel=0: 종료 후 관련 동영상 숨기기
            - modestbranding=1: YouTube 로고 최소화 
        - UI 관련
            - controls=0: 컨트롤바 숨기기
            - fs=0: 전체화면 버튼 숨기기 
        - 자막
            - cc_load_policy=1: 자막 기본 표시 

        - 파라미터 조합 방법
            - 첫번째 파라미터: ?로 시작
            - 추가 파라미터: &로 연결

    4) 반응형 비디오 컨테이너 CSS 기법 습득 
        - 반응형 원리 (padding hack)
            - padding의 %는 "부모 너비" 기준으로 계산됨
                - 너비 : 높이 = 16:9

            - 우리가 알고 있는 것 : 너비 (width: 100%)
            - 우리가 구해야 하는 것 : 높이 (padding-bottom: ?%)

            - 비율 공식 : 높이 / 너비 = 9 / 16 = 0.5625
            => 높이는 너비의 56.26% 
               예) 800px * 0.5625 = 450px (높이)

        - 비율         계산           padding-bottom
          --------------------------------------------------
          16:9       9 / 16            56.25% (YouTube 기본)
          4:3        3 / 4             75% (구형TV)
          21:9       9 / 21            42.86% (영화 와이드)
          1:1        1 / 1             100% (정사각형)
