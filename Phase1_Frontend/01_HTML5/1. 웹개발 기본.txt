1. HTML 기본 문서 만들기 
    1) HTML
        - HyperText Markup Language
        - 웹 문서를 만드는 언어 
        - HyperText : 문서를 서로 연결해주는 링크 
    2) HTML 기본 문서 구조 
        - <!DOCTYPE html>
            - 현재 문서가 HTML 언어로 작성한 웹 문서라는 뜻임.
            - 현재 표준은 HTML5
        - <html> ~ </html>
            - 웹 문서의 시작과 끝을 나타내는 태그
        - <head> ~ </head>
            - 웹 브라우저가 웹 문서를 해석하는데 필요한 정보를 입력하는 부분임. 
            - <meta>
                - HTML 문서(웹페이지)에 관한 정보(제작자,내용,키워드 등)를 검색엔진이나 브라우저에게 제공
        - <body> ~ </body>
            - 실제로 웹 브라우저 화면에 나타나는 내용임   

2. Viewport 메타 태그 가이드 
    1) <meta name="viewport" content="width=device-width, inital-scale=1.0">
        - 모든 웹 프로젝트에 기본으로 들어가야 함 
        - HTML 템플릿, 프레임워크(React, Vue 등) 모두 기본으로 포함되어 있음

    2) 반응형 웹 구현
        - 먼저 viewport 메타 태그로 기기 너비에 맞추고, CSS 미디이쿼리로 화면 크기별 스타일 적용함   

    3) viewport
        - 브라우저에서 웹 컨텐츠가 표시되는 가시 영역   

    4) width=device-width
        - 디바이스 화면 너비를 viewport 너비로 사용

    5) inital-scale=1.0
        - 페이지 로드 시 1:1 비율로 표시 (확대/축소 없음)
